Cash Manager

Manage who owes who.

Pranesh Srinivasan
License : GPL

Description

A program that allows you to keep track of whom you owe, and who owes you how much - in two hundred haskell lines!

Download it here.

Dependencies

A good haskell compiler. I have tested this with the Glasgow Haskell Compiler (GHC) on a GNU/Linux Machine (Ubuntu). It should also work on Windows, using ghc.

Usage

The program allows the user to have buddies, that he declares using cashman add friend. The user can then keep track individually of the money he gives or takes from someone.

Right now, the program is first person centric.

Examples : 
[spranesh@nemesis:~]$ cashman add smith                          (01-01 15:43)
Thu Jan 1 2009 :: User smith Created

[spranesh@nemesis:~]$ cashman take smith 5                       (01-01 15:44)
Thu Jan 1 2009 :: took 5  smith

[spranesh@nemesis:~]$ cashman give smith 100 "lunch"             (01-01 15:44)
Thu Jan 1 2009 :: gave 100  lunch  smith

[spranesh@nemesis:~]$ cashman report                             (01-01 15:44)
Each number shown is your asset with that person. In other words, a negative
amount indicates that you owe the person, and a positive that the person owes
you.
